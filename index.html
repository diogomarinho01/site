Churn Preditivo Telecom

Desenvolver um modelo capaz de prever churn de clientes de telecom a partir de uma base disponibilizada pela IBM. Neste projeto, utilizamos a linguagem estatistica R (RStudio) e disponibilizamos o resultado no Shiny Web Server. Começamos com carga e limpeza dos dados, depois partirmos para a Análise Exploratória dos Dados e, na sequência, Experimentamos os modelos de Regressão Logistica, Análise de Variância, Arvore de decisão e RandomForest em busca daquele que nos dá o melhor percentual de assertividade.
